About This Project

This community forum was created as a group project for our SDEV265 class at Ivy Tech.

Our goal was to build a functional, user-friendly web application where people can share ideas, post discussions, and connect with others in a meaningful way. We used Django for the backend and Bootstrap for styling.

This project allowed us to practice real-world skills like web development, collaboration, version control, and project planning.
